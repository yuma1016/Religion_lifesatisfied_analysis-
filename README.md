# Religion_lifesatisfied_analysis-
# 宗教と生活満足度のデータ分析（World Values Survey）

## 概要
このプロジェクトは、World Values Survey (WVS) のデータを用いて、各国の「神の重要度」と「生活満足度」の関係を分析するためのRスクリプトです。単回帰・重回帰分析による統計的な検証と、地図やグラフを用いた可視化を行っています。

## 使用データ
本スクリプトを実行するには、WVSの第7波（Wave 7）のデータが必要です。
* **ファイル名**: `WVS_Cross-National_Wave_7_rds_v6_0.rds`
* **入手方法**: [World Values Survey 公式サイト](https://www.worldvaluessurvey.org/) からダウンロードし、スクリプトと同じディレクトリ（フォルダ）内に配置してください。
* ※データサイズおよびライセンスの都合上、データファイル本体はこのリポジトリには含んでいません。

## 必要なRパッケージ
スクリプトの実行には以下のパッケージが必要です。事前にインストールをお願いします。
* `ggplot2`
* `dplyr`
* `haven`
* `sf`
* `rnaturalearth`
* `rnaturalearthdata`
* `viridis`

## 使い方
1. WVSのデータ（`.rds`）をダウンロードし、プロジェクトフォルダに保存します。
2. RStudio等で `int_microdata_2425716.R` を開きます。
3. コードの最初の行にあるデータ読み込み部分のファイルパスが正しいか確認し、全体を実行します。

## 出力される図表
コードを実行すると以下の結果が出力されます。
* **図1**: 国別の平均的な「神の重要度」のコロプレス図（地図）
* **図2**: 生活満足度のヒストグラム
* **図3**: 神の重要度と生活満足度の散布図（主要宗教ごとの色分け）
* **図4**: 宗教圏ごとの生活満足度の箱ひげ図
* **表1**: 生活満足度を応答変数とした単回帰および重回帰分析結果
